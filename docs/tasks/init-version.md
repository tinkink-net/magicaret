# Magicaret Init Version

Magicaret is a lib that turn input caret into a custom caret with beautiful style and smooth animation.

## Goal

- Multi target libs
  - Vanilla JS
  - React
  - Vue
- Easy to use
  - Simple API
  - No dependency
- Beautiful style
  - Default gradient color
  - Has several built-in styles, covers the major colors
  - Customizable style such as color, size, animation, etc.
- Smooth animation
  - It animates the caret position smoothly
  - With default effects when it moves, just like the comet with a trail

## Requirements

- Use Vite & rolldown-vite as bundler
- Use pnpm as package manager
- Use TypeScript
- Support ES module and CommonJS module for final output
- Bundle as npm package
- Comprehensive documentation & examples
