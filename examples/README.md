# Magicaret Examples

This directory contains examples demonstrating the Magicaret library in different frameworks.

## Examples

### Vanilla JS

A simple HTML/JavaScript example showing how to use Magicaret with vanilla JavaScript.

**Run it:**
```bash
cd examples/vanilla
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### React

A React example using Vite that demonstrates the `MagicaretInput` component.

**Run it:**
```bash
cd examples/react
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

### Vue

A Vue 3 example using Vite that demonstrates the `MagicaretInput` component.

**Run it:**
```bash
cd examples/vue
npm install
npm run dev
```

Then open http://localhost:3001 in your browser.

## Features Demonstrated

- All examples showcase:
  - Multiple built-in styles (default, neon, rainbow, fire, ocean, purple, green, pink)
  - Smooth caret animations with comet trail effect
  - Real-time style switching
  - Different input types (text, email, textarea)

## Customization

Each example allows you to:
1. Select different caret styles
2. Type in various input fields
3. See the animated caret in action
4. Experience the smooth animation and comet trail effect

Try typing and moving the cursor around to see the beautiful animations!
